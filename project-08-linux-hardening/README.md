# Linux Hardening Lab (Pro)

A Linux security project focused on system hardening and security checks.

## Features
- System update
- Disable insecure services
- Enable firewall (UFW)
- Secure SSH configuration
- File permission hardening
- Security audit script

## Files
- hardening.sh → apply security settings
- check.sh → verify system security

## How to Run
```bash
chmod +x hardening.sh
chmod +x check.sh

./hardening.sh
./check.sh
