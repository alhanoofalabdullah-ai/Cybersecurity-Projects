# Secure SSH Access Auditor

A professional cybersecurity project focused on auditing SSH access security, detecting weak configurations, unauthorized access risks, and improving secure remote administration practices.

---

## Overview

This project helps system administrators strengthen Linux server security by reviewing SSH settings and identifying dangerous access configurations.

---

## Features

- Audit SSH configuration files
- Detect weak SSH settings
- Identify root login risks
- Check password authentication status
- Improve secure remote access
- Security monitoring automation

---

## Project Structure

- main.py → Main SSH audit script
- ssh_config.txt → Sample SSH configuration file
- risky_settings.txt → Dangerous SSH settings list
- requirements.txt → Python dependencies
- commands.txt → Setup commands
- README.md → Project documentation
