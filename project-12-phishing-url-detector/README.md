# Phishing URL Detector

A professional cybersecurity project focused on detecting suspicious URLs using basic security indicators and Python.

---

## Overview

This project analyzes URLs and identifies possible phishing patterns such as suspicious keywords, insecure protocols, long URLs, and unusual domain structures.

---

## Features

- Check suspicious keywords
- Detect insecure HTTP URLs
- Identify long URLs
- Detect excessive special characters
- Generate risk score
- Display security result

---

## Project Structure

- main.py → Main detection script
- requirements.txt → Python dependencies
- commands.txt → Setup commands
- README.md → Project documentation
