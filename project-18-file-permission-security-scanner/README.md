# File Permission Security Scanner

A professional cybersecurity project focused on scanning file permissions and detecting risky or insecure file access settings using Python.

---

## Overview

This project checks file permissions inside a selected directory and identifies files with dangerous permissions such as world-writable access.

---

## Features

- Scan file permissions
- Detect world-writable files
- Identify insecure permission settings
- Security monitoring automation
- Fast filesystem inspection
- Permission security reporting

---

## Project Structure

- main.py → Main scanner script
- scan_folder/ → Example folder to scan
- requirements.txt → Python dependencies
- commands.txt → Setup commands
- README.md → Project documentation

## Technologies Used

- File Permission Analysis
- Linux Security Basics
- Security Monitoring
