# Password Leak Checker

A professional cybersecurity project focused on checking whether passwords exist in known leaked password databases using Python.

---

## Overview

This project helps identify weak or compromised passwords by comparing user input against a list of commonly leaked passwords.

---

## Features

- Password leak detection
- Weak password identification
- Compare against leaked password database
- Security awareness improvement
- Fast password validation
- Cybersecurity automation

---

## Project Structure

- main.py → Main password checker script
- leaked_passwords.txt → Known leaked passwords
- requirements.txt → Python dependencies
- commands.txt → Setup commands
- README.md → Project documentation
