# Security Log Alert System

A professional cybersecurity project focused on monitoring security logs and generating alerts for suspicious activities using Python.

---

## Overview

This project scans security logs for dangerous keywords such as failed login attempts, brute force indicators, and unauthorized access attempts.

---

## Features

- Monitor security logs
- Detect suspicious login attempts
- Identify brute force attacks
- Unauthorized access detection
- Automated alert generation
- Fast security monitoring

---

## Project Structure

- main.py → Main monitoring script
- security_logs.txt → Sample security logs
- alert_keywords.txt → Dangerous keywords list
- requirements.txt → Python dependencies
- commands.txt → Setup commands
- README.md → Project documentation
